# 🌌 AETHER AI — Visual Synthesis Platform

Aether AI is a full-stack neural image synthesis application featuring an interactive multi-stage generation workflow, real-time status HUD, AI-powered prompt enhancement, custom style matrices, and a session history gallery.

---

## 🎨 Features

* **🪄 Magic AI Prompt Enhancer:** Automatically expands simple user prompts into rich, highly detailed cinematic prompts using automated descriptor enrichment logic.
* **⚙️ Stage 1 — Parameter Control Studio:** Customizable prompt input, artistic style matrices (Cyberpunk, Realism, Anime, 3D Render), and aspect ratio controls (512px to 1024px).
* **👁️ Stage 2 — Output Viewport:** Real-time generation progress indicator, full-screen canvas preview, and high-resolution image downloads.
* **🔄 Workflow Pipeline Visualizer:** Top-level HUD banner demonstrating real-time stages from prompt ingestion to neural execution.
* **🕒 Session Gallery:** Automatically saves and displays generated artworks for quick previewing and reuse during the active session.

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite), Framer Motion, FontAwesome, Glassmorphic CSS Engine
* **Backend:** Python 3, Flask, Flask-CORS, Requests, Gunicorn
* **Inference Engine:** Pollinations AI (FLUX Model Core)
